O que é "html"?

'HyperText Markup Language', html. é uma linguagem de marcação fundamental para o processo de criação das pagínas Web. Em poucas palavras, seria aonde montariamos o "esqueleto" de nosso site, para que depois viessemos estilizando juntamente com o css.

Criado por Tim Berners-Lee (físico britânico), criador dessa linguagem de marcação, hoje ela é de extrema importância na codificação dos Websites. Devido ao tipo "linguagem", ela é bastante confundida com linguagem de programação, havendo bastante diferença entre as duas.
